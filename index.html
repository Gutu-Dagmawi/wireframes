Patient Journey Flows

1. Clinical Visit (Practitioner Encounter)
Receptionist
Register patient
Check in
Select practitioner
→
Finance
Card activation
payment
Must pay before queue
→
Nurse
Vitals & triage
BP, temp, pulse, etc.
→
Practitioner
Assessment, diagnosis
Lab orders, prescriptions
Order-sheets, referrals
Lab Order Path
Practitioner → Finance (pay) → Lab Tech (in-house) → Results → Practitioner
External tests: Practitioner generates referral directly
Order-Sheet Path
Practitioner → Finance (pay) → Nurse (administer) → Completed → Practitioner
Encounter Finalized
Practitioner → Receptionist (Encounter Outputs) → Print → Patient
Prescriptions, referral letters, certificates
Encounter Suspended
Practitioner suspends → Receptionist (Suspended Encounters) → Patient returns
Resumption: Receptionist activates → Nurse (re-triage) → Practitioner (Resumed queue)
For external lab results taking days/weeks. Card valid until practitioner's declared limit.
2. Special Service (Walk-In, No Doctor)
Receptionist
Create special
service order
BP, blood sugar, injection…
→
Finance
Service payment
Auto-priced from admin
→
Nurse / Lab Tech
Perform service
or process specimen
→
Receptionist
Results routed back
Print & deliver
No practitioner involved
Result Routing Key
■ Practitioner-ordered lab → results to Practitioner
■ Special service lab → results to Receptionist
■ External-only test → Practitioner generates referral (not lab tech)
■ Encounter outputs → prescriptions/referrals to Receptionist for printing
■ Suspended encounter → Receptionist manages resumption (walk-in or appointment)
